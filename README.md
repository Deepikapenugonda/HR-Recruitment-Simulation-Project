# HR Recruitment Simulation Project

## Project Overview
End-to-end recruitment sourcing and screening simulation
for HR roles at Swiggy, Zepto and Groww — Bangalore.

## Tools Used
Google Sheets
Google Docs
LinkedIn
Naukri

## What This Project Covers
Candidate sourcing across 3 channels
Resume screening with scoring rubric
Pipeline tracking for 15 candidates
Dashboard with 4 charts and key metrics
Sourcing report with findings and recommendations

## Key Metrics
Total Candidates Sourced: 15
Shortlisting Rate: 87%
Top Source Channel: LinkedIn
Companies Covered: Swiggy, Zepto, Groww
